Hello. I'm an interaction designer who cooks a lot. So I noticed a whole lot of ergonomic inefficiencies in the most popular timer apps. Since so many people use timer apps every day, I decided to address them.

Outside of side-projects like this, I spend most of my time R&Ding social technology. My position is that many of the problems with the web are a caused by the technologies we've built it on top of, which put too much burden on producers of content while not giving users enough agency. A better web would make it much easier for users to, for instance, build experimental extensions to shared software. This will require new languages, new databases, new protocols. Many other problems with the web are simply interface design problems, which I'm working on today.

My other output can be viewed here: [aboutmako.makopool.com](https://aboutmako.makopool.com)
